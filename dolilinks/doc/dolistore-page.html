DoliLinks - Advanced Company Link Management

Create and manage hierarchical relationships between companies in Dolibarr

Hierarchical Link Management

Create parent-child relationships between companies with customizable link types (subsidiary, branch, partner, etc.).

Interactive Visualization

Interactive network diagram showing all relationships with color-coded nodes and clickable navigation.

Smart Contact Filtering

Intelligent filtering of billing contacts to prevent sending invoices to your clients' clients.

Flexible Configuration

Customizable link types, multi-entity support, and seamless Dolibarr integration.

Key Features

  • Company Link Management: Create parent-child relationships between companies with customizable link types
  • Hierarchical Visualization: Automatic display in company cards with interactive network diagram
  • Advanced Contact Management: Smart filtering of billing contacts and child company contact display
  • Data Import & Migration: SocParent module migration tool with data validation
  • Multi-language Support: French, English, German, Spanish, Italian
  • Dolibarr Integration: Native hooks and seamless integration with existing workflows

Technical Specifications

Compatibility: Dolibarr 20.0+
Languages: French, English, German, Spanish, Italian
Multi-entity: Full support
Security: Integrated with Dolibarr security system
Architecture: MVC with hooks system
License: GPLv3
