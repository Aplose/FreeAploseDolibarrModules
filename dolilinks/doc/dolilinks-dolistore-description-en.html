DoliLinks - Hierarchical Links Management Between Companies

DoliLinks is an innovative module for Dolibarr that allows you to create and manage hierarchical links between companies (third parties). It provides a clear visualization of parent-child relationships between businesses and facilitates the management of complex organizational structures with an interactive diagram.

Main features

🔗 Company Link Management

Hierarchical relationships

  • Create parent-child links between companies
  • Customizable link types (subsidiary, branch, partner, etc.)
  • Circular link prevention
  • Centralized management via administration interface

Display in company cards

  • "Parents" section: List of parent companies with direct links
  • "Children" section: List of child companies with direct links
  • Quick action buttons: Add links and access diagram
  • Automatic counters: Number of parents/children displayed

📊 Interactive Diagram

Complete graphical visualization

  • Hierarchical network: Display of all parents, children and grandchildren
  • Interactive navigation: Click on nodes to access company cards
  • Color legend: Visual distinction between parents (gray), current company (green) and children (blue)
  • Link types: Display of link type labels on connections

👥 Smart Contact Filtering

Billing contact management

  • Automatic filtering: Only propose billing contacts when sending emails
  • Linked company contacts: Display of child company contacts in cards
  • Flexible configuration: Enable/disable via module settings
  • Avoid errors: Don't send invoices to your clients' clients!

⚙️ Link Type Management

Complete customization

  • Custom type creation: Define specific relationship types
  • Centralized management: Integrated administration interface
  • Dolibarr dictionary: Storage in standard dictionary
  • Predefined types: Subsidiary, branch, partner, etc.

Use cases

  • Corporate groups: Manage relationships between parent company and subsidiaries
  • Commercial networks: Visualize relationships between partners and distributors
  • Complex structures: Understand complex hierarchical relationships
  • Centralized billing: Manage billing for multiple linked entities
  • Consolidated reporting: Analyze performance of a group of companies

Advanced features

  • Import from SocParent: Migration tool to import data from SocParent module
  • Multi-entity: Full support for Dolibarr multi-entity mode
  • Security: Respect for access rights and Dolibarr security
  • Translations: Multilingual support (French, English, German, Spanish, Italian)
