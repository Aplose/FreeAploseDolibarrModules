DoliLinks - Gestion des Liens Hiérarchiques entre Sociétés

DoliLinks est un module innovant pour Dolibarr qui permet de créer et gérer des liens hiérarchiques entre les sociétés (tiers). Il offre une visualisation claire des relations parent-enfant entre les entreprises et facilite la gestion des structures organisationnelles complexes avec un diagramme interactif.

Fonctionnalités principales

🔗 Gestion des Liens entre Sociétés

Relations hiérarchiques

  • Création de liens parent-enfant entre sociétés
  • Types de liens personnalisables (filiale, succursale, partenaire, etc.)
  • Prévention des liens circulaires
  • Gestion centralisée via interface d'administration

Affichage dans les fiches

  • Section "Parents" : Liste des sociétés parentes avec liens directs
  • Section "Enfants" : Liste des sociétés filles avec liens directs
  • Boutons d'action rapides : Ajout de liens et accès au diagramme
  • Compteurs automatiques : Nombre de parents/enfants affiché

📊 Diagramme Interactif

Visualisation graphique complète

  • Réseau hiérarchique : Affichage de tous les parents, enfants et petits-enfants
  • Navigation interactive : Clic sur les nœuds pour accéder aux fiches sociétés
  • Légende colorée : Distinction visuelle entre parents (gris), société courante (vert) et enfants (bleu)
  • Types de liens : Affichage des labels des types de liens sur les connexions

👥 Filtrage Intelligent des Contacts

Gestion des contacts de facturation

  • Filtrage automatique : Ne proposer que les contacts de facturation lors de l'envoi d'emails
  • Contacts des sociétés liées : Affichage des contacts des sociétés enfants dans les fiches
  • Configuration flexible : Activation/désactivation via les paramètres du module
  • Évite les erreurs : Ne pas envoyer la facture aux clients de vos clients !

⚙️ Gestion des Types de Liens

Personnalisation complète

  • Création de types personnalisés : Définir des types de relations spécifiques
  • Gestion centralisée : Interface d'administration intégrée
  • Dictionnaire Dolibarr : Stockage dans le dictionnaire standard
  • Types prédéfinis : Filiale, succursale, partenaire, etc.

Cas d'usage

  • Groupes d'entreprises : Gérer les relations entre maison mère et filiales
  • Réseaux commerciaux : Visualiser les relations entre partenaires et distributeurs
  • Structures complexes : Comprendre les relations hiérarchiques complexes
  • Facturation centralisée : Gérer la facturation pour plusieurs entités liées
  • Reporting consolidé : Analyser les performances d'un groupe d'entreprises

Fonctionnalités avancées

  • Import depuis SocParent : Outil de migration pour importer les données du module SocParent
  • Multi-entité : Support complet du mode multi-entité Dolibarr
  • Sécurité : Respect des droits d'accès et de la sécurité Dolibarr
  • Traductions : Support multilingue (français, anglais, allemand, espagnol, italien)
