DoliCarbon - Bilans d’émissions de gaz à effet de serre (GES)

Le module DoliCarbon permet de structurer et suivre un inventaire d’émissions de gaz à effet de serre directement dans Dolibarr : bilans par exercice, lignes d’activité (scopes 1, 2 et 3), facteurs d’émission, actions de réduction, cadrage méthodologique, tableaux de bord, rapports avec exports, et assistant d’import depuis vos données Dolibarr. Une interface web intégrée (Angular) complète les écrans classiques du module.

Pourquoi choisir DoliCarbon ?

DoliCarbon centralise l’inventaire carbone dans Dolibarr : mêmes utilisateurs, mêmes droits d’accès, données liées à votre référentiel (tiers, achats, etc.) lorsque vous utilisez l’import et les options de rappel à la validation. Les scopes 1, 2 et 3 sont pris en charge avec une nomenclature de catégories explicite, des facteurs d’émission traçables et des écrans de restitution (exécutif, analyste, annexe méthodologique).

Fonctionnalités principales

🌿 Bilans d’inventaire

Périodes et références

  • Inventaires par année avec référence automatique du type CARBON-{année}-{séquence}
  • Dates de début et de fin de période, lien optionnel vers un tiers
  • Totaux et objectifs en tCO2e, statuts de workflow (brouillon, validé, archivé)

📊 Lignes d’activité

Scopes et calcul

  • Saisie par scope 1, 2 ou 3 et par catégories du module
  • Quantités, unités, liaison à un facteur d’émission pour le calcul des émissions
  • Workflow de validation, commentaires et traçabilité ; liaison possible à des objets sources (ex. factures fournisseurs importées)

🔢 Facteurs d’émission

  • Bibliothèque de facteurs d’émission (données de versionnement selon le modèle)
  • Activation ou désactivation selon les règles prévues par le schéma

🎯 Actions de réduction

  • Actions rattachées à un bilan : gains estimés, coûts, scores, statut de suivi

📐 Cadrage méthodologique

  • Documentation du périmètre, exclusions, matérialité, années de référence et de reporting
  • Textes réutilisés dans la partie rapport (annexe méthodologique)

📈 Tableau de bord et rapports

  • Vue agrégée : indicateurs, scopes, série temporelle, catégories dominantes
  • Rapport multi-volets : synthèse exécutive, détail analyste, annexe méthodologique
  • Exports CSV et JSON ; instantané figé (snapshot) pour un état reproductible
  • Texte de garde-fou de communication configurable lorsque renseigné

✅ Qualité et traçabilité

  • Suivi de workflow sur les lignes, commentaires, journal d’audit

🔄 Plan de transition

  • Vue synthétique des actions de réduction pour le pilotage (coût / bénéfice selon l’interface)

⬇️ Import depuis Dolibarr

  • Assistant d’import pour rapprocher des données Dolibarr de vos lignes d’inventaire

⚙️ Options administrateur

Rappels à la validation (paramétrables)

  • À la validation d’une facture fournisseur, d’une note de frais ou d’une expédition : messages d’information vers DoliCarbon si l’option est activée (lien vers l’import pour les factures fournisseurs)
  • Import automatique optionnel : à la validation d’une facture fournisseur, création d’une ligne scope 3 (catégorie purchases_services) sous conditions : bilan en brouillon, facteur actif adapté, montant HT en EUR ; les doublons sont évités par un hash d’import

🖥️ Deux modes d’accès

  • Interface intégrée (/custom/dolicarbon/index.php) : tableau de bord, bilans, cadrage, lignes, facteurs, actions, qualité, transition, import, rapport (droit dolicarbon : lecture minimum)
  • Écrans PHP : liste des bilans, facteurs, et autres pages carbon_*.php fournies avec le module

Avantages clés

  • Inventaire dans Dolibarr : un seul socle pour les droits et les données
  • Scopes 1, 2 et 3 avec catégories explicites
  • Calcul traçable : lignes reliées aux facteurs d’émission
  • Rapports et exports : CSV, JSON, snapshot
  • Interface moderne : application Angular (Material) pour l’app intégrée
  • Droits granulaires : lecture, écriture, suppression, validation (workflow)
  • Langues module Dolibarr : français (fr_FR), anglais (en_US) ; interface Angular avec fichier fr.json

Cas d’usage

  • RSE et QHSE : tenir un inventaire annuel et suivre la réduction des émissions
  • PME et ETI : centraliser bilans, facteurs et actions sans tableur isolé
  • Cabinets et consultants : structurer le suivi pour plusieurs entités (multicompany selon configuration Dolibarr)
  • Lien achats / logistique : rappels et import optionnel depuis les factures fournisseur validées

Option 2 : Installation manuelle

  1. Téléchargez le module depuis DoliStore
  2. Déployez-le sous htdocs/custom/dolicarbon
  3. Activez le module dans la configuration Dolibarr
  4. Configurez les options (rappels à la validation, import automatique facture fournisseur) dans la page de configuration du module
  5. Attribuez les droits DoliCarbon aux groupes d’utilisateurs
