DoliCarbon - Greenhouse gas (GHG) emission inventories

The DoliCarbon module lets you build and track a greenhouse gas emission inventory inside Dolibarr: year-based inventories, activity lines (scopes 1, 2 and 3), emission factors, reduction actions, methodological framing, dashboards, reporting with exports, and an import wizard from your Dolibarr data. An embedded web application (Angular) complements the standard module screens.

Why choose DoliCarbon?

DoliCarbon keeps your carbon inventory in Dolibarr: same users, same permissions, and data connected to your master data (third parties, purchases, etc.) when you use import and validation hints. Scopes 1, 2 and 3 are supported with explicit categories, traceable emission factors, and reporting views (executive, analyst, methodological annex).

Main features

🌿 Inventory periods (“bilans”)

Periods and references

  • Year-based inventories with automatic references such as CARBON-{year}-{sequence}
  • Start and end dates, optional link to a third party
  • Totals and targets in tCO2e, workflow status (draft, validated, archived)

📊 Activity lines

Scopes and calculation

  • Entry by scope 1, 2 or 3 and by module categories
  • Quantities, units, link to an emission factor for emission calculation
  • Validation workflow, comments and traceability; possible link to source objects (e.g. imported supplier invoices)

🔢 Emission factors

  • Emission factor library (versioning fields in the data model)
  • Activate or deactivate factors according to the schema rules

🎯 Reduction actions

  • Actions attached to an inventory: estimated savings, costs, scores, status

📐 Methodological framing

  • Document scope, exclusions, materiality, reference and reporting years
  • Texts reused in the report (methodological annex)

📈 Dashboard and reports

  • Aggregated view: indicators, scopes, time series, main categories
  • Multi-part report: executive summary, analyst detail, methodological annex
  • CSV and JSON export; snapshot for a reproducible state
  • Optional communication guardrail text when configured

✅ Quality and traceability

  • Workflow on lines, comments, audit log

🔄 Transition plan

  • Summary view of reduction actions for steering (cost / benefit style in the UI)

⬇️ Import from Dolibarr

  • Import wizard to map Dolibarr data to inventory lines

⚙️ Administrator options

Validation hints (configurable)

  • When a supplier invoice, expense report or shipment is validated: informational messages toward DoliCarbon if enabled (link to import for supplier invoices)
  • Optional automatic import: when a supplier invoice is validated, creates a scope 3 line (category purchases_services) under conditions: draft inventory, matching active factor, amount excl. tax in EUR; duplicates avoided by an import hash

🖥️ Two access modes

  • Embedded UI (/custom/dolicarbon/index.php): dashboard, inventories, framing, lines, factors, actions, quality, transition, import, report (minimum dolicarbon read permission)
  • PHP screens: inventory list, factors, and other shipped carbon_*.php pages

Key advantages

  • Inventory inside Dolibarr: one place for permissions and data
  • Scopes 1, 2 and 3 with explicit categories
  • Traceable calculation: lines linked to emission factors
  • Reports and exports: CSV, JSON, snapshot
  • Modern UI: Angular (Material) for the embedded app
  • Granular rights: read, write, delete, validate (workflow)
  • Module languages: French (fr_FR), English (en_US); embedded UI uses fr.json

Use cases

  • CSR / HSE: run an annual inventory and track emission reductions
  • SMEs and mid-caps: centralize inventories, factors and actions without a standalone spreadsheet
  • Consultancies: structure tracking for several entities (multicompany per Dolibarr setup)
  • Purchasing / logistics: hints and optional import from validated supplier invoices

Option 2: Manual installation

  1. Download the module from DoliStore
  2. Deploy it under htdocs/custom/dolicarbon
  3. Enable the module in Dolibarr configuration
  4. Set options (validation hints, automatic supplier-invoice import) in the module setup page
  5. Assign DoliCarbon permissions to user groups
