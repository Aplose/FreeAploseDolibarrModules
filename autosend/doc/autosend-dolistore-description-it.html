AutoSend - Invio Automatico di File via FTP

Il modulo AutoSend automatizza l'invio dei vostri file generati (CSV, ODS) verso server FTP/FTPS. Ideale per l'integrazione con i vostri sistemi esterni, funziona in complemento al modulo Ods2Csv per automatizzare i vostri flussi documentali.

Funzionalità principali

📤 Invio Automatico via FTP/FTPS

Trasmissione sicura

  • Invio automatico di file verso server FTP o FTPS (SSL)
  • Supporto di connessioni sicure FTPS
  • Configurazione flessibile per regola
  • Gestione di directory remote

Regole di corrispondenza

  • Filtraggio per espressione regolare (regex) sui nomi di file
  • Attivazione in base allo stato dell'oggetto Dolibarr
  • Attivazione/disattivazione per regola
  • Gestione di più regole simultanee

Rinomina automatica

  • Rinomina dei file con pattern dinamici
  • Variabili disponibili: #REF#, #REF_CUSTOMER#, #YYYY#, #MM#, #DD#, ecc.
  • Personalizzazione completa dei nomi di file remoti

Casi d'uso

  • Integrazione EDI: Invio automatico di file verso sistemi partner
  • Esportazione contabile: Trasmissione automatica verso software contabili
  • Archiviazione: Backup automatico su server remoti
  • Workflow documentale: Automatizzazione dei flussi di documenti
