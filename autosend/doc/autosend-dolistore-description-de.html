AutoSend - Automatischer Dateiversand via FTP

Das Modul AutoSend automatisiert den Versand Ihrer generierten Dateien (CSV, ODS) an FTP/FTPS-Server. Ideal für die Integration mit Ihren externen Systemen, funktioniert es ergänzend zum Modul Ods2Csv zur Automatisierung Ihrer Dokumentenflüsse.

Hauptfunktionen

📤 Automatischer FTP/FTPS-Versand

Sichere Übertragung

  • Automatischer Dateiversand an FTP- oder FTPS- (SSL) Server
  • Unterstützung für sichere FTPS-Verbindungen
  • Flexible Konfiguration pro Regel
  • Verwaltung von Remote-Verzeichnissen

Zuordnungsregeln

  • Filterung nach regulärem Ausdruck (Regex) auf Dateinamen
  • Auslösung basierend auf Dolibarr-Objektstatus
  • Aktivierung/Deaktivierung pro Regel
  • Verwaltung mehrerer gleichzeitiger Regeln

Automatische Umbenennung

  • Dateiumbenennung mit dynamischen Mustern
  • Verfügbare Variablen: #REF#, #REF_CUSTOMER#, #YYYY#, #MM#, #DD#, etc.
  • Vollständige Anpassung der Remote-Dateinamen

Anwendungsfälle

  • EDI-Integration: Automatischer Dateiversand an Partner-Systeme
  • Buchhaltungsexport: Automatische Übertragung an Buchhaltungssoftware
  • Archivierung: Automatisches Backup auf Remote-Server
  • Dokumenten-Workflow: Automatisierung von Dokumentenflüssen
