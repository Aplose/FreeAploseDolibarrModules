AutoSend - Envoi Automatique de Fichiers par FTP

Le module AutoSend automatise l'envoi de vos fichiers générés (CSV, ODS) vers des serveurs FTP/FTPS. Idéal pour l'intégration avec vos systèmes externes, il fonctionne en complément du module Ods2Csv pour automatiser vos flux documentaires.

Fonctionnalités principales

📤 Envoi Automatique par FTP/FTPS

Transmission sécurisée

  • Envoi automatique de fichiers vers serveurs FTP ou FTPS (SSL)
  • Support des connexions sécurisées FTPS
  • Configuration flexible par règle
  • Gestion des répertoires distants

Règles de correspondance

  • Filtrage par expression régulière (regex) sur les noms de fichiers
  • Déclenchement selon le statut de l'objet Dolibarr
  • Activation/désactivation par règle
  • Gestion de multiples règles simultanées

Renommage automatique

  • Renommage des fichiers avec patterns dynamiques
  • Variables disponibles : #REF#, #REF_CUSTOMER#, #YYYY#, #MM#, #DD#, etc.
  • Personnalisation complète des noms de fichiers distants

Cas d'usage

  • Intégration EDI : Envoi automatique de fichiers vers systèmes partenaires
  • Export comptable : Transmission automatique vers logiciels comptables
  • Archivage : Sauvegarde automatique sur serveurs distants
  • Workflow documentaire : Automatisation des flux de documents
