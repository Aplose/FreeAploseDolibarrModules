AutoSend - Automatic File Sending via FTP

The AutoSend module automates sending your generated files (CSV, ODS) to FTP/FTPS servers. Ideal for integration with your external systems, it works in complement with the Ods2Csv module to automate your document flows.

Main features

📤 Automatic FTP/FTPS Sending

Secure transmission

  • Automatic file sending to FTP or FTPS (SSL) servers
  • Support for secure FTPS connections
  • Flexible configuration per rule
  • Remote directory management

Matching rules

  • Filtering by regular expression (regex) on file names
  • Triggering based on Dolibarr object status
  • Enable/disable per rule
  • Multiple simultaneous rules management

Automatic renaming

  • File renaming with dynamic patterns
  • Available variables: #REF#, #REF_CUSTOMER#, #YYYY#, #MM#, #DD#, etc.
  • Complete customization of remote file names

Use cases

  • EDI Integration: Automatic file sending to partner systems
  • Accounting export: Automatic transmission to accounting software
  • Archiving: Automatic backup to remote servers
  • Document workflow: Document flow automation
