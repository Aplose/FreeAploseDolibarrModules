Ods2Csv - Conversion Automatique ODS vers CSV

Ods2Csv est un module pratique qui convertit automatiquement les fichiers ODS (LibreOffice Calc) en fichiers CSV dès qu'un fichier ODS est généré depuis un modèle dans Dolibarr. Idéal pour automatiser l'export de données et l'intégration avec d'autres systèmes.

Fonctionnalités principales

🔄 Conversion Automatique

Génération automatique de CSV

  • Conversion automatique des fichiers ODS en CSV lors de la génération
  • Extraction de la première feuille du fichier ODS
  • Conversion via LibreOffice en ligne de commande
  • Aucune intervention manuelle requise
  • Intégration transparente avec les modèles ODS de Dolibarr

⚙️ Configuration Flexible

Séparateur personnalisable

  • Séparateur par défaut : virgule (,)
  • Personnalisation du séparateur via paramètre ODS2CSVSEPARATOR
  • Configuration dans "Configuration > Divers"
  • Support de tous les caractères séparateurs standards
  • Encodage UTF-8 par défaut

🔗 Intégration avec AutoSend

Automatisation complète

  • Compatible avec le module AutoSend pour envoi FTP automatique
  • Génération et envoi automatiques des fichiers CSV
  • Workflow complet : ODS → CSV → FTP
  • Renommage dynamique des fichiers selon règles définies
  • Intégration transparente dans vos processus métier

📊 Cas d'Usage

Exports et intégrations

  • Export de données pour systèmes externes (ERP, WMS, etc.)
  • Génération de fichiers d'expédition (fiches d'expédition CSV)
  • Export de commandes pour systèmes logistiques
  • Intégration avec systèmes de facturation tiers
  • Export de données comptables

Cas d'usage

  • Logistique : Génération automatique de fichiers CSV pour les systèmes WMS ou de transport
  • Facturation : Export automatique des données de facturation vers systèmes externes
  • Comptabilité : Export des écritures comptables en format CSV
  • Expéditions : Génération de fiches d'expédition au format CSV pour les transporteurs
  • Intégration ERP : Export de données pour synchronisation avec autres ERP
  • Automatisation : Workflow complet avec AutoSend pour envoi automatique par FTP

Prérequis

  • Dolibarr ERP & CRM installé (version 18.0 ou supérieure)
  • LibreOffice installé sur le serveur (pour la conversion)
  • Configuration MAIN_ODT_AS_PDF = 'libreoffice' dans Dolibarr
  • Modèles ODS configurés dans Dolibarr
