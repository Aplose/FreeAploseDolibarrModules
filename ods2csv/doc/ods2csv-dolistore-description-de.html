Ods2Csv - Automatische ODS zu CSV-Konvertierung

Ods2Csv ist ein praktisches Modul, das ODS-Dateien (LibreOffice Calc) automatisch in CSV-Dateien konvertiert, sobald eine ODS-Datei aus einer Vorlage in Dolibarr generiert wird. Ideal für die Automatisierung des Datenexports und die Integration mit anderen Systemen.

Hauptfunktionen

🔄 Automatische Konvertierung

Automatische CSV-Generierung

  • Automatische Konvertierung von ODS-Dateien in CSV während der Generierung
  • Extraktion des ersten Blatts aus der ODS-Datei
  • Konvertierung über LibreOffice-Befehlszeile
  • Kein manueller Eingriff erforderlich
  • Nahtlose Integration mit Dolibarr ODS-Vorlagen

⚙️ Flexible Konfiguration

Anpassbares Trennzeichen

  • Standardtrennzeichen: Komma (,)
  • Trennzeichen-Anpassung über Parameter ODS2CSVSEPARATOR
  • Konfiguration in "Einstellungen > Sonstige Einstellungen"
  • Unterstützung für alle Standard-Trennzeichen
  • UTF-8-Kodierung standardmäßig

🔗 Integration mit AutoSend

Vollständige Automatisierung

  • Kompatibel mit AutoSend-Modul für automatischen FTP-Versand
  • Automatische Generierung und Versendung von CSV-Dateien
  • Vollständiger Workflow: ODS → CSV → FTP
  • Dynamische Umbenennung von Dateien nach definierten Regeln
  • Nahtlose Integration in Ihre Geschäftsprozesse

📊 Anwendungsfälle

Exporte und Integrationen

  • Datenexport für externe Systeme (ERP, WMS, etc.)
  • Generierung von Versanddateien (CSV-Versandblätter)
  • Bestell-Export für Logistiksysteme
  • Integration mit Drittanbieter-Abrechnungssystemen
  • Buchhaltungsdatenexport

Anwendungsfälle

  • Logistik: Automatische Generierung von CSV-Dateien für WMS- oder Transportsysteme
  • Abrechnung: Automatischer Export von Abrechnungsdaten zu externen Systemen
  • Buchhaltung: Export von Buchungssätzen im CSV-Format
  • Versand: Generierung von Versandblättern im CSV-Format für Spediteure
  • ERP-Integration: Datenexport zur Synchronisation mit anderen ERPs
  • Automatisierung: Vollständiger Workflow mit AutoSend für automatischen FTP-Versand

Voraussetzungen

  • Dolibarr ERP & CRM installiert (Version 18.0 oder höher)
  • LibreOffice auf dem Server installiert (für Konvertierung)
  • MAIN_ODT_AS_PDF = 'libreoffice' Konfiguration in Dolibarr
  • ODS-Vorlagen in Dolibarr konfiguriert
