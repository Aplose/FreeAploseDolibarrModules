Ods2Csv - Conversión Automática ODS a CSV

Ods2Csv es un módulo práctico que convierte automáticamente archivos ODS (LibreOffice Calc) a archivos CSV tan pronto como se genera un archivo ODS desde una plantilla en Dolibarr. Ideal para automatizar la exportación de datos y la integración con otros sistemas.

Funcionalidades principales

🔄 Conversión Automática

Generación automática de CSV

  • Conversión automática de archivos ODS a CSV durante la generación
  • Extracción de la primera hoja del archivo ODS
  • Conversión mediante línea de comandos de LibreOffice
  • No se requiere intervención manual
  • Integración perfecta con plantillas ODS de Dolibarr

⚙️ Configuración Flexible

Separador personalizable

  • Separador por defecto: coma (,)
  • Personalización del separador mediante parámetro ODS2CSVSEPARATOR
  • Configuración en "Configuración > Diversos"
  • Soporte de todos los caracteres separadores estándar
  • Codificación UTF-8 por defecto

🔗 Integración con AutoSend

Automatización completa

  • Compatible con el módulo AutoSend para envío FTP automático
  • Generación y envío automáticos de archivos CSV
  • Flujo de trabajo completo: ODS → CSV → FTP
  • Renombrado dinámico de archivos según reglas definidas
  • Integración perfecta en sus procesos empresariales

📊 Casos de Uso

Exportaciones e integraciones

  • Exportación de datos para sistemas externos (ERP, WMS, etc.)
  • Generación de archivos de envío (hojas de envío CSV)
  • Exportación de pedidos para sistemas logísticos
  • Integración con sistemas de facturación de terceros
  • Exportación de datos contables

Casos de uso

  • Logística: Generación automática de archivos CSV para sistemas WMS o de transporte
  • Facturación: Exportación automática de datos de facturación a sistemas externos
  • Contabilidad: Exportación de asientos contables en formato CSV
  • Envíos: Generación de hojas de envío en formato CSV para transportistas
  • Integración ERP: Exportación de datos para sincronización con otros ERP
  • Automatización: Flujo de trabajo completo con AutoSend para envío automático por FTP

Prerrequisitos

  • Dolibarr ERP & CRM instalado (versión 18.0 o superior)
  • LibreOffice instalado en el servidor (para conversión)
  • Configuración MAIN_ODT_AS_PDF = 'libreoffice' en Dolibarr
  • Plantillas ODS configuradas en Dolibarr
