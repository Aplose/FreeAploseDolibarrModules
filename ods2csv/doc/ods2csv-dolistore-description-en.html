Ods2Csv - Automatic ODS to CSV Conversion

Ods2Csv is a practical module that automatically converts ODS files (LibreOffice Calc) to CSV files as soon as an ODS file is generated from a template in Dolibarr. Ideal for automating data export and integration with other systems.

Main features

🔄 Automatic Conversion

Automatic CSV generation

  • Automatic conversion of ODS files to CSV during generation
  • Extraction of the first sheet from the ODS file
  • Conversion via LibreOffice command line
  • No manual intervention required
  • Seamless integration with Dolibarr ODS templates

⚙️ Flexible Configuration

Customizable separator

  • Default separator: comma (,)
  • Separator customization via ODS2CSVSEPARATOR parameter
  • Configuration in "Setup > Other Setup"
  • Support for all standard separator characters
  • UTF-8 encoding by default

🔗 Integration with AutoSend

Complete automation

  • Compatible with AutoSend module for automatic FTP sending
  • Automatic generation and sending of CSV files
  • Complete workflow: ODS → CSV → FTP
  • Dynamic file renaming according to defined rules
  • Seamless integration into your business processes

📊 Use Cases

Exports and integrations

  • Data export for external systems (ERP, WMS, etc.)
  • Shipping file generation (CSV shipping sheets)
  • Order export for logistics systems
  • Integration with third-party billing systems
  • Accounting data export

Use cases

  • Logistics: Automatic generation of CSV files for WMS or transport systems
  • Billing: Automatic export of billing data to external systems
  • Accounting: Export of accounting entries in CSV format
  • Shipping: Generation of shipping sheets in CSV format for carriers
  • ERP integration: Data export for synchronization with other ERPs
  • Automation: Complete workflow with AutoSend for automatic FTP sending

Prerequisites

  • Dolibarr ERP & CRM installed (version 18.0 or higher)
  • LibreOffice installed on the server (for conversion)
  • MAIN_ODT_AS_PDF = 'libreoffice' configuration in Dolibarr
  • ODS templates configured in Dolibarr
