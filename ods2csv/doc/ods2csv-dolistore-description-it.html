Ods2Csv - Conversione Automatica ODS in CSV

Ods2Csv è un modulo pratico che converte automaticamente i file ODS (LibreOffice Calc) in file CSV non appena viene generato un file ODS da un modello in Dolibarr. Ideale per automatizzare l'esportazione dei dati e l'integrazione con altri sistemi.

Funzionalità principali

🔄 Conversione Automatica

Generazione automatica CSV

  • Conversione automatica di file ODS in CSV durante la generazione
  • Estrazione del primo foglio del file ODS
  • Conversione tramite riga di comando LibreOffice
  • Nessun intervento manuale richiesto
  • Integrazione perfetta con modelli ODS Dolibarr

⚙️ Configurazione Flessibile

Separatore personalizzabile

  • Separatore predefinito: virgola (,)
  • Personalizzazione separatore tramite parametro ODS2CSVSEPARATOR
  • Configurazione in "Configurazione > Altri parametri"
  • Supporto per tutti i caratteri separatori standard
  • Codifica UTF-8 predefinita

🔗 Integrazione con AutoSend

Automazione completa

  • Compatibile con modulo AutoSend per invio FTP automatico
  • Generazione e invio automatici di file CSV
  • Flusso di lavoro completo: ODS → CSV → FTP
  • Rinomina dinamica file secondo regole definite
  • Integrazione perfetta nei vostri processi aziendali

📊 Casi d'Uso

Esportazioni e integrazioni

  • Esportazione dati per sistemi esterni (ERP, WMS, ecc.)
  • Generazione file spedizione (fogli spedizione CSV)
  • Esportazione ordini per sistemi logistici
  • Integrazione con sistemi fatturazione terze parti
  • Esportazione dati contabili

Casi d'uso

  • Logistica: Generazione automatica file CSV per sistemi WMS o trasporto
  • Fatturazione: Esportazione automatica dati fatturazione verso sistemi esterni
  • Contabilità: Esportazione scritture contabili in formato CSV
  • Spedizioni: Generazione fogli spedizione in formato CSV per corrieri
  • Integrazione ERP: Esportazione dati per sincronizzazione con altri ERP
  • Automazione: Flusso di lavoro completo con AutoSend per invio automatico FTP

Prerequisiti

  • Dolibarr ERP & CRM installato (versione 18.0 o superiore)
  • LibreOffice installato sul server (per conversione)
  • Configurazione MAIN_ODT_AS_PDF = 'libreoffice' in Dolibarr
  • Modelli ODS configurati in Dolibarr
