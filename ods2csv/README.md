# ODS2CSV FOR [DOLIBARR ERP CRM](https://www.dolibarr.org)

## Features

Onse activated, Ods2Csv allow you to generate automaticaly a csv file from the first sheet of any ods file generated from an ods template.

To change the field separator (, by default), you need to add in "Setup->Other Setup" a new parameter ODS2CSVSEPARATOR with the choosen character. Enjoy !

Other external modules are available on [Dolistore.com](https://www.dolistore.com).

## Translations

Translations can be completed manually by editing files into directories _langs_.
