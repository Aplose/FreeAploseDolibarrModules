
Directory for module image files
--------------------------------

You can put here the .png files of your module:


If the picto of your module is an image (property $picto has been set to 'productrecall.png@productrecall', you can put into this
directory a .png file called *object_productrecall.png* (16x16 or 32x32 pixels)


If the picto of an object is an image (property $picto of the object.class.php has been set to 'myobject.png@productrecall', then you can put into this
directory a .png file called *object_myobject.png* (16x16 or 32x32 pixels)
