ProductRecall - Suivi des Rappels Produits

Surveillez les alertes conso et protégez vos clients avec les rappels produits officiels

Surveillance Automatique

Surveillance automatique des rappels produits officiels avec alertes par email personnalisables.

Filtrage Intelligent

Filtrage par catégories et sous-catégories de produits pour recevoir uniquement les alertes pertinentes.

Interface Moderne

Interface moderne et intuitive avec recherche instantanée, navigation fluide et affichage en cartes.

Documents Officiels

Accès direct aux documents PDF officiels et aux affichettes à imprimer pour vos points de vente.

Intégration Produits

Onglet dédié dans les fiches produits pour consulter directement les rappels liés à chaque produit.

Fonctionnalités Principales

  • Surveillance Automatique : Récupération automatique des rappels depuis le site RappelConso officiel
  • Alertes Email : Notifications automatiques par email avec documents PDF joints
  • Filtrage Personnalisé : Configuration de filtres par catégories et sous-catégories de produits
  • Interface Moderne : Interface utilisateur moderne avec recherche instantanée et navigation fluide
  • Widget d'Accueil : Affichage des derniers rappels directement sur la page d'accueil Dolibarr
  • Intégration Produits : Onglet dédié dans les fiches produits pour consulter les rappels liés
  • Documents Officiels : Téléchargement direct des affichettes PDF et fiches de rappel
  • Gestion Complète : Consultation de tous les rappels actuels et passés avec détails complets

⚠️ Prérequis Important

Ce module nécessite l'installation préalable du module AploseFramework (gratuit) disponible sur DoliStore. L'AploseFramework est indispensable au bon fonctionnement de l'interface moderne.

Spécifications Techniques

Compatibilité : Dolibarr 20.0+

Installation et Support

Installation Facile

  1. Installez d'abord le module AploseFramework (gratuit) depuis DoliStore
  2. Téléchargez le module ProductRecall depuis DoliStore
  3. Uploadez via l'installateur de modules Dolibarr
  4. Activez dans la section modules
  5. Configurez vos filtres de catégories et adresses email
  6. Activez le job de surveillance automatique
  7. Commencez à recevoir vos alertes !
