ChangeModelPDFLocation - Modifica Percorsi Modelli PDF

ChangeModelPDFLocation è un modulo di amministrazione che vi permette di modificare in massa i percorsi dei modelli PDF nel vostro database Dolibarr. Essenziale durante una migrazione, cambio di ubicazione dei modelli o riorganizzazione della vostra struttura di file.

Funzionalità principali

📁 Modifica in Massa dei Percorsi PDF

Aggiornamento automatico dei percorsi

  • Sostituzione automatica di una parte del percorso nei campi model_pdf
  • Aggiornamento di tutte le tabelle interessate in una sola operazione
  • Supporto di più tipi di documenti: ordini, fatture, proposte, spedizioni
  • Visualizzazione del numero di righe modificate per tabella

Tabelle aggiornate

  • Ordini clienti: Aggiornamento del modello PDF degli ordini
  • Ordini fornitori: Aggiornamento del modello PDF degli ordini fornitori
  • Fatture: Aggiornamento del modello PDF delle fatture
  • Proposte commerciali: Aggiornamento del modello PDF delle proposte
  • Spedizioni: Aggiornamento del modello PDF delle spedizioni
  • File ECM: Aggiornamento dei percorsi dei file documentali

Configurazione semplice

  • Definizione del percorso vecchio da sostituire
  • Definizione del nuovo percorso di destinazione
  • Esecuzione in un clic
  • Report dettagliato delle modifiche effettuate

Casi d'uso

  • Migrazione server: Adattare i percorsi dopo un cambio di server o hosting
  • Riorganizzazione: Spostare i vostri modelli PDF in una nuova ubicazione
  • Correzione percorsi: Correggere percorsi errati nel database
  • Standardizzazione: Uniformare i percorsi dei modelli PDF
  • Migrazione multi-entità: Adattare i percorsi durante una migrazione multi-entità

⚠️ Modulo Amministratore

ChangeModelPDFLocation è esclusivamente destinato agli amministratori Dolibarr. Solo gli utenti con diritti amministratore possono modificare i percorsi dei modelli PDF. Si raccomanda di fare un backup del database prima di qualsiasi modifica.
