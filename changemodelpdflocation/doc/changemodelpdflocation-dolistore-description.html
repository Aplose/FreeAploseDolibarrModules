ChangeModelPDFLocation - Modification des Chemins des Modèles PDF

ChangeModelPDFLocation est un module d'administration qui permet de modifier en masse les chemins des modèles PDF dans votre base de données Dolibarr. Indispensable lors d'une migration, d'un changement d'emplacement des modèles ou d'une réorganisation de votre structure de fichiers.

Fonctionnalités principales

📁 Modification en Masse des Chemins PDF

Mise à jour automatique des chemins

  • Remplacement automatique d'une partie du chemin dans les champs model_pdf
  • Mise à jour de toutes les tables concernées en une seule opération
  • Support de multiples types de documents : commandes, factures, propositions, expéditions
  • Affichage du nombre de lignes modifiées par table

Tables mises à jour

  • Commandes clients : Mise à jour du modèle PDF des commandes
  • Commandes fournisseurs : Mise à jour du modèle PDF des commandes fournisseurs
  • Factures : Mise à jour du modèle PDF des factures
  • Propositions commerciales : Mise à jour du modèle PDF des propositions
  • Expéditions : Mise à jour du modèle PDF des expéditions
  • Fichiers ECM : Mise à jour des chemins des fichiers documentaires

Configuration simple

  • Définition de l'ancien chemin à remplacer
  • Définition du nouveau chemin de destination
  • Exécution en un clic
  • Rapport détaillé des modifications effectuées

Cas d'usage

  • Migration de serveur : Adapter les chemins après un changement de serveur ou d'hébergement
  • Réorganisation : Déplacer vos modèles PDF vers un nouvel emplacement
  • Correction de chemins : Corriger des chemins incorrects dans la base de données
  • Standardisation : Uniformiser les chemins de modèles PDF
  • Migration multi-entités : Adapter les chemins lors d'une migration multi-entités

⚠️ Module Administrateur

ChangeModelPDFLocation est exclusivement destiné aux administrateurs Dolibarr. Seuls les utilisateurs ayant les droits administrateur peuvent modifier les chemins des modèles PDF. Il est recommandé de faire une sauvegarde de la base de données avant toute modification.
