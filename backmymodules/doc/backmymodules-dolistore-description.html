Fonctionnalités principales

💾 Sauvegarde Complète de vos Modules

Archive ZIP de tous vos modules

  • Création d'une archive ZIP contenant tous les modules du répertoire /custom/
  • Téléchargement direct de l'archive pour sauvegarde locale
  • Protection contre la perte de modules (lien Dolistore expiré, prestataire indisponible)
  • Préparation sécurisée des mises à jour Dolibarr

Gestion des modules désactivés

  • Liste de tous les modules complémentaires installés
  • Suppression sécurisée des modules désactivés uniquement
  • Préservation des données (répertoire "documents" non supprimé)
  • Nettoyage pour éviter les conflits lors des mises à jour

Cas d'usage

  • Reprise de contrôle : Récupérer vos modules si votre prestataire n'est plus disponible
  • Préparation de mise à jour : Sauvegarder vos modules avant une mise à jour majeure de Dolibarr
  • Nettoyage : Supprimer les anciens modules désactivés qui peuvent causer des problèmes
  • Migration : Transférer vos modules vers une nouvelle instance Dolibarr
  • Archivage : Conserver une copie de sécurité de tous vos modules personnalisés

⚠️ Module Administrateur

BackMyModules est exclusivement destiné aux administrateurs Dolibarr. Seuls les utilisateurs ayant les droits administrateur peuvent accéder aux fonctionnalités de sauvegarde et de suppression de modules.
