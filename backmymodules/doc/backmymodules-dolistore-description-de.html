Hauptfunktionen

💾 Vollständiges Backup Ihrer Module

ZIP-Archiv aller Ihrer Module

  • Erstellung eines ZIP-Archivs mit allen Modulen im Verzeichnis /custom/
  • Direkter Download des Archivs für lokales Backup
  • Schutz vor Modulverlust (abgelaufener Dolistore-Link, nicht verfügbarer Anbieter)
  • Sichere Vorbereitung von Dolibarr-Updates

Verwaltung deaktivierter Module

  • Liste aller installierten Add-on-Module
  • Sichere Löschung nur deaktivierter Module
  • Datenerhaltung (Verzeichnis "documents" wird nicht gelöscht)
  • Bereinigung zur Vermeidung von Konflikten während Updates

Anwendungsfälle

  • Kontrolle zurückgewinnen: Ihre Module wiederherstellen, wenn Ihr Anbieter nicht mehr verfügbar ist
  • Update-Vorbereitung: Backup Ihrer Module vor einem wichtigen Dolibarr-Update
  • Bereinigung: Alte deaktivierte Module löschen, die Probleme verursachen können
  • Migration: Ihre Module auf eine neue Dolibarr-Instanz übertragen
  • Archivierung: Eine Backup-Kopie aller Ihrer benutzerdefinierten Module behalten

⚠️ Administratormodul

BackMyModules ist ausschließlich für Dolibarr-Administratoren bestimmt. Nur Benutzer mit Administratorrechten können auf die Backup- und Modullöschfunktionen zugreifen.
