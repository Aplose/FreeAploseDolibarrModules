Funcionalidades principales

💾 Respaldo Completo de sus Módulos

Archivo ZIP de todos sus módulos

  • Creación de un archivo ZIP que contiene todos los módulos del directorio /custom/
  • Descarga directa del archivo para respaldo local
  • Protección contra pérdida de módulos (enlace Dolistore expirado, proveedor no disponible)
  • Preparación segura de actualizaciones de Dolibarr

Gestión de módulos desactivados

  • Lista de todos los módulos complementarios instalados
  • Eliminación segura solo de módulos desactivados
  • Preservación de datos (directorio "documents" no eliminado)
  • Limpieza para evitar conflictos durante las actualizaciones

Casos de uso

  • Recuperación de control: Recuperar sus módulos si su proveedor ya no está disponible
  • Preparación de actualización: Hacer respaldo de sus módulos antes de una actualización importante de Dolibarr
  • Limpieza: Eliminar módulos antiguos desactivados que pueden causar problemas
  • Migración: Transferir sus módulos a una nueva instancia de Dolibarr
  • Archivado: Mantener una copia de seguridad de todos sus módulos personalizados

⚠️ Módulo Administrador

BackMyModules está exclusivamente destinado a administradores de Dolibarr. Solo los usuarios con derechos de administrador pueden acceder a las funcionalidades de respaldo y eliminación de módulos.
