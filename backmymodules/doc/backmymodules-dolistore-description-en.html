Main features

💾 Complete Backup of Your Modules

ZIP archive of all your modules

  • Create a ZIP archive containing all modules in the /custom/ directory
  • Direct download of the archive for local backup
  • Protection against module loss (expired Dolistore link, unavailable provider)
  • Safe preparation of Dolibarr updates

Management of deactivated modules

  • List of all installed add-on modules
  • Safe deletion of deactivated modules only
  • Data preservation (documents directory not deleted)
  • Cleanup to avoid conflicts during updates

Use cases

  • Regaining control: Recover your modules if your provider is no longer available
  • Update preparation: Backup your modules before a major Dolibarr update
  • Cleanup: Delete old deactivated modules that can cause problems
  • Migration: Transfer your modules to a new Dolibarr instance
  • Archiving: Keep a backup copy of all your custom modules

⚠️ Administrator Module

BackMyModules is exclusively intended for Dolibarr administrators. Only users with administrator rights can access the backup and module deletion features.
